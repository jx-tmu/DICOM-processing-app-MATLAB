# DICOM医学影像处理与特征提取App

## 项目简介
本项目为基于MATLAB App Designer开发的医学影像处理应用，支持DICOM及常见图片格式的浏览、处理、分割与特征提取。

## 功能概览
- 选择并浏览DICOM或图片文件夹，自动排序影像序列
- 图像处理：均值滤波、中值滤波、高斯滤波、阈值分割（手动/大津）、聚类分割、边缘检测
- 区域生长分割：支持多点种子交互式选取
- ROI特征提取：面积、质心、YOLO框、周长、偏心率、主轴/副轴等
- 数据保护：防止ROI特征区被随意修改

## 使用方法
1. 启动App，点击“选择文件夹”选择影像文件夹
2. 使用滑动条浏览不同序列影像
3. 选择右侧处理按钮进行图像处理
4. 区域增长：点击“区域增长”，在图像上选取种子点，在对话框选择是否继续添加种子结束。
5. 点击“计算”提取分割区域特征

## 依赖环境
- MATLAB R2021b
- Image Processing Toolbox
- 支持App Designer

- 支持格式：.dcm, .jpg, .png, .bmp, .tif等
